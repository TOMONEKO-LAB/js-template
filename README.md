# Dev Template

個人開発で使用する（した）開発環境を構築できるCLIツールです．

そのままクローンするだけで使えます．

使い方等はまとめておきますので，使えそうであれば使ってみてください．

# 概要

個人開発をしていく上で，開発をするときの環境が __マルチプラットフォーム__ であったり，一度経験したことがある __フレームワーク__ であったりと，割と固定化されていました．

その都度環境をコピペして作ったりするのが煩わしく感じましたので __テンプレート化__ します．

# 前提

この環境を使用するうえで，前提となっている環境があるのでまとめます．

以下の環境がご用意されていると使えます．

- `Git` を使用できる
- `Docker` を使用できる
- `VSCode` を使用できる
- `VSCode` で拡張機能 `Dev Containers` を使用できる

# 使い方

## 1. セットアップ

初めは，必要に応じて以下コマンドを実行してください．

```bash
docker compose up set-up
```

もともとプロジェクトを作成してある方は，Dockerfileと同階層にプロジェクトを展開してください．

実行されると，環境が作成されます．

## 2. パッケージの取得

本環境は`npm`を使用しています．

別のパッケージを使用している場合は，`docker-compose.yml`の __30__ 行目のコマンドを書き換えてください．

```bash
docker compose up pkgs
```

## 3. 開発サーバー（ホットリロード付き）の起動

開発サーバーを起動します．

```bash
docker compose up --watch app
```
